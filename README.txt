This is the raycast kernel of the SLAMBench benchmark. 
SLAMBench is a computer vision benchmark containing a total of 14 kernels, references: 
1) http://arxiv.org/abs/1410.2167
2) http://apt.cs.manchester.ac.uk/projects/PAMELA/tools/SLAMBench/

To run the application from a shell do: 
$ make
